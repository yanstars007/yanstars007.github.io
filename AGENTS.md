# AGENTS.md

## Project Overview
- Single-file static app: `index.html`
- Vue 3 (CDN), no build tooling, no package manager
- Login page with CSS animations (petals, hearts, sparkles)
- Demo credentials: `admin` / `123456`
- Auth state stored in `localStorage`

## How to Run
- Open `index.html` directly in a browser, or use any static server (e.g. `npx serve .`)

## Conventions
- All code (HTML, CSS, JS) lives in `index.html`
- `.history/` directory is gitignored (VS Code history plugin)
- Chinese UI labels and validation messages
